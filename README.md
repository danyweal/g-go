# Palestinian Community Association App

## Overview
A Next.js app for the Palestinian Community Association in the North West (UK). Features: news, events, donations (Stripe), join community form (emailed via SendGrid), contact, about.

## Setup
1. Clone repo
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env.local` and fill in secrets
4. Run dev server: `npm run dev`

## Environment Variables
See `.env.example` for required variables.

## Scripts
- `npm run dev` - local development
- `npm run build` - production build
- `npm run start` - start built app

## Deployment
Recommend Vercel. Set environment variables there.
